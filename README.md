<h2>Running</h2>
<br>
<b>Clone the project<b>
<br>
<code>git clone https://github.com/otavosousa/calendar-app.git</code>
<br>
<br>
<b>Install expo<b>
<br>
<code>npm install --global expo-cli</code>
<br>
<br>
<b>Install dependences<b>
<br>
<code>npm install</code>
<br>
<br>
<b>Run<b>
<br>
<code>npm start</code>
